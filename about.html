How this website works

It uses the github REST api which is a public API and fetches the repo list of the given user, Then that responce is parsed to get the repo name, progamming language used and the repo link.Then it is then displayed on list with a generated link button from the repo list


The Website is made with the following technologies
  • HTML
  • Bootstrap 4
  • JavaScript
  • Github REST API
